What is the differences between innerHTML,innerText and textContent?
textContent is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.
What is the Event Delegation in DOM?
Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element.
What is the Event Bubbling in DOM?
The process of propagating events from the target element up to its parent elements in the DOM hierarchy.
